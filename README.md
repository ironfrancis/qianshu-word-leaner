# 英语单词打字练习程序

基于艾宾浩斯遗忘曲线的英语单词学习打字练习工具。

## 功能特点

- 📚 **多单词源支持**: 支持小学英语单词1、2、3（共448个单词）
- 🎯 **打字练习**: 显示中文释义，输入对应英文单词
- 🧠 **艾宾浩斯记忆曲线**: 科学安排复习间隔
- 📊 **学习统计**: 追踪正确率、学习进度
- ❌ **错题本**: 自动记录错误单词，重点练习
- 💾 **进度保存**: 自动保存学习进度到浏览器

## 使用方法

### Windows 用户

1. 双击运行 `start.bat` 启动本地服务器
2. 浏览器会自动打开 http://localhost:8000
3. 选择要学习的单词源
4. 开始打字练习！

### 手动启动

如果自动启动失败，请手动操作：

1. 打开命令提示符
2. 进入项目目录：`cd english-typing-practice`
3. 运行服务器：`python -m http.server 8000`
4. 浏览器访问：http://localhost:8000

## 练习模式

### 📖 学习模式
- 学习新单词和需要复习的单词
- **新单词学习**: 第一次遇到新单词时，会自动先显示答案（中文→英文）让你学习3秒，然后开始练习
- **跳过处理**: 跳过的单词会被记录为"不会"，加入错题本
- **智能提示系统**:
  - 停止输入 **8秒** 后，开始逐字母提示
  - 每 **5秒** 显示一个新字母
  - 卡住 **30秒** 后，显示完整学习卡片
  - **使用提示的单词算"学习中"**，不会进入"已掌握"状态

### 🔄 复习模式
- 根据艾宾浩斯遗忘曲线安排复习
- 自动筛选到期需要复习的单词

### ❌ 错题本
- 专门练习错误率高的单词
- 按错误率从高到低排序

### 🔄 复习模式
- 根据艾宾浩斯遗忘曲线安排复习
- 自动筛选到期需要复习的单词

### ❌ 错题本
- 专门练习错误率高的单词
- 按错误率从高到低排序

## 艾宾浩斯复习间隔

| 阶段 | 复习间隔 |
|------|----------|
| 1 → 2 | 5分钟 |
| 2 → 3 | 30分钟 |
| 3 → 4 | 12小时 |
| 4 → 5 | 1天 |
| 5 → 6 | 2天 |
| 6 → 7 | 4天 |
| 7 → 8 | 7天 |
| 8 → 9 | 15天 |

## 键盘快捷键

- `Enter` - 显示答案 / 下一个单词
- `Tab` - 跳过当前单词
- `Esc` - 返回选择页面

## 数据说明

单词数据来自：
- `data/9.xml` - 小学英语单词1（200词）
- `data/10.xml` - 小学英语单词2（200词）
- `data/11.xml` - 小学英语单词3（48词）
- `data/trans_dict.xml` - 英汉翻译字典

## 学习进度

学习进度自动保存在浏览器 localStorage 中，包括：
- 每个单词的复习阶段
- 正确/错误次数
- 最后复习时间

如需清除进度，按 F12 打开浏览器控制台，输入：
```javascript
localStorage.clear()
```
